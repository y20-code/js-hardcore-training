# 业务需求文档 (PRD)：武器铁匠铺锻造功能

## 业务背景
在“Word Killer”游戏的城镇里，有一个铁匠铺。玩家可以拿着初始武器和一堆材料去找铁匠升级武器。你需要构建这个铁匠铺的底层系统。

## 研发任务
创建一个纯对象 `forgeSystem`，包含以下内容：

### 1. 店铺基础信息
* `smithName`: "欧冶子"
* `shopLevel`: 1

### 2. 核心动作：升级武器 (`upgradeWeapon`)
**输入要求**：
* 第一个参数：武器的基础攻击力（必传，数字）。
* 第二个参数：锻造手续费。如果没传，**默认为 50 金币**。
* 剩余参数：一堆强化材料的能量值（如 10, 20...）。全部打包成数组。

**内部计算**：
* 算出所有材料的总能量值（没材料则为 0）。
* 最终攻击力 = 基础攻击力 + 材料总能量值 + (店铺等级 * 10)。**必须读取对象自身的等级**。

**输出交付**：
* 返回对象：包含 `finalAttack` (最终攻击力) 和 `cost` (手续费)。

### 3. 核心动作：全服广播 (`broadcastAd`)
* 无需参数和返回值。
* 触发后**等待 2 秒**。
* 打印：“我是 欧冶子，我的铁匠铺目前是 1 级，欢迎来打造神器！”（需动态读取自身属性）。
